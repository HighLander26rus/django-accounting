# Задание №1: CRUD для справочника «Клиенты»
## Цель
Научиться создавать простые CRUD-интерфейсы на Django с использованием generic views и ModelForm.
## Исходные данные
- Модель `Klient` уже создана в `inventory/models.py`. - Админка Django настроена (вы можете посмотреть структуру полей). - В базе уже есть 
несколько клиентов (загружены тестовыми данными).
## Что нужно сделать
Создать страницы для управления списком клиентов: 1. **Список клиентов** (`/clients/`) – таблица с колонками:
   - Имя
   - Флаг «Это я» (да/нет)
   - Кнопки действий: Редактировать, Удалить 2. **Добавление клиента** (`/clients/add/`) – форма с полями:
   - Имя (обязательное)
   - Флаг «Это я» (чекбокс, по умолчанию False) 3. **Редактирование клиента** (`/clients/<id>/edit/`) – форма, предзаполненная данными 
выбранного клиента. 4. **Удаление клиента** (`/clients/<id>/delete/`) – страница подтверждения удаления.
## Технические требования
- Использовать **class-based views**: `ListView`, `CreateView`, `UpdateView`, `DeleteView`. - Формы должны наследоваться от `ModelForm`. - 
После успешного добавления/редактирования/удаления – редирект на список клиентов. - Для удаления использовать `DeleteView` с шаблоном 
подтверждения. - URL-маршруты должны быть в файле `inventory/urls.py` (если его нет – создайте) и подключены в корневом `config/urls.py`. - 
Шаблоны должны наследовать базовый шаблон (можно создать `templates/base.html` с минимальной структурой и подключить Bootstrap для 
удобства).
## Ожидаемый результат
- При переходе на `/clients/` отображается таблица клиентов. - Доступны все CRUD операции. - Интерфейс выглядит аккуратно (использование 
Bootstrap приветствуется, но не обязательно).
## Подсказки
- Примеры generic views: [Django documentation](https://docs.djangoproject.com/en/5.1/topics/class-based-views/generic-display/) - Для 
маршрутов используйте `path()`. - Не забудьте добавить приложение `inventory` в `INSTALLED_APPS` (уже добавлено). - Для работы со 
статическими файлами (CSS) настройте `STATIC_URL` и `STATICFILES_DIRS` (в проекте уже настроено, но вы можете добавить свою статику).
