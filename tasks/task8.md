# Задание №8: Отчёт P&L (прибыли и убытки)
## Цель
Научиться строить агрегированные отчёты на основе транзакций, группировать данные по типам операций, рассчитывать финансовые показатели.
## Исходные данные
- Модель `Transaction` с типами операций: `sale_rev` (выручка), `sale_cost` (себестоимость продаж), `expense` (прочие расходы). - В базе 
есть транзакции разных типов за разные периоды. - Константы `REVENUE_GUID` и `COST_OF_SALES_GUID` (используются для фильтрации, но можно 
фильтровать по `operation_type`).
## Что нужно сделать
Создать страницу для отображения отчёта о прибылях и убытках за выбранный период. **Страница:** `/reports/pl/` **Форма фильтра:** - Период: 
начальная дата и конечная дата (обязательные поля). - (Опционально) Выбор клиента/товара/другой аналитики – но для базовой версии 
достаточно дат. **Отчёт должен содержать:**
| Показатель | Сумма | ------------|-------| Выручка (sale_rev) | сумма | Себестоимость продаж (sale_cost) | сумма | **Валовая прибыль** 
| (Выручка - Себестоимость) | сумма | Прочие расходы (expense) | сумма | **Чистая прибыль** (Валовая прибыль - Прочие расходы) | сумма |
## Технические требования
- Использовать агрегацию через `aggregate` и `Sum` для каждого типа операций. - Для получения выручки и себестоимости можно фильтровать 
`Transaction` по `operation_type` и дате. - Если в периоде нет операций какого-то типа, сумма должна быть 0 (не `None`). - Выводить даты 
периода в заголовке отчёта.
## Ожидаемый результат
- Пользователь выбирает даты, нажимает «Показать». - Отчёт показывает суммы выручки, себестоимости, валовой прибыли, прочих расходов и 
чистой прибыли за указанный период. - Все суммы отображаются в рублях (с двумя знаками после запятой).
## Подсказки
- Пример запроса:
  ```python
  from django.db.models import Sum
  from inventory.models import Transaction
  revenue = Transaction.objects.filter(
      operation_type='sale_rev',
      date__range=[start_date, end_date]
  ).aggregate(total=Sum('amount'))['total'] or 0
```
- Аналогично для sale_cost и expense.
- Валовая прибыль = revenue - cogs.
- Чистая прибыль = gross_profit - expenses.
- В шаблоне можно использовать простую таблицу.
